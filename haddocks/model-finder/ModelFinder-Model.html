class Model k a model | model -> k a where Source #

model represents possible assignments φ :: k -> a.

  • An assignment can be partial
  • A model can represent a priori knowledge about the assignments (e.g. injectivity "type" restriction naturality / etc.)

Minimal complete definition

guess, guessMany, enterDef

Methods

guess :: k -> model -> [a] Source #

Get the current possible assignments. Returning singleton means the value of k is already settled. It can be empty if the model has gotten inconsistent.

guessMany :: NonEmpty k -> model -> [a] Source #

Get the intersection of guesses for the given multiple queries.

enterDef :: [k] -> a -> model -> Maybe (model, [(k, a)]) Source #

Update the model accoring to newly added definitive equality φ(k1) = φ(k2) = ... = y :: a. Returns the updated model and newly discovered definitive equalities. Returns Nothing if the given equality is incompatible with model.

enterEqs :: [k] -> model -> Maybe (model, [(k, a)]) Source #

Update the model accoring to newly added equality φ(k1) = φ(k2) = ... = φ(kn). Returns the updated model and newly discovered definitive equalities. Returns Nothing if the given equality is incompatible with model.

data SimpleModel k a Source #
