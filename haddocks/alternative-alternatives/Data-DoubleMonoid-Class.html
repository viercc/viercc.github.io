class DoubleMonoid a where Source #

DoubleMonoid is a type with two independent monoid structures.

The two monoids are named "additive" (zero, /+/) and "multiplicative" (one, /*/) respectively. But unlike Num or (semi)ring structures, DoubleMonoid assume no relation between the two monoids like distributivity.

Laws

Monoid laws of "additive" set of operators:

(x /+/ y) /+/ z === x /+/ (y /+/ z)
zero /+/ x === x
x /+/ zero === x

Monoid laws of "multiplicative" set of operators:

(x /*/ y) /*/ z === x /*/ (y /*/ z)
one /*/ x === x
x /*/ one === x

Relation to other known algebraic structures

Of course, DoubleMonoid can be seen as Monoid in two ways. Other than that, near-semiring (TODO:url) is a subclass of DoubleMonoid which additionally satisfy "left distribution law" (which confusingly have another name "right distributivity").

-- in near-semiring
zero /*/ x  === zero
(x /+/ y) /*/ z === (x /*/ z) /+/ (y /*/ z)

Minimal complete definition

one, (/*/), zero, (/+/) | mprod, msum
