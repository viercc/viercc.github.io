class LZ a => LZLC a where Source #

TwoMonoids with left zero (LZ) and additional left catch property: one is right-absorbing element for the additive monoid.

-- left catch
one <+> x === one

msumZ :: SumZ a -> a Source #

msumZ is a ZList algebra.

msumZ . fmap msumZ === msumZ . join

When you override the default implementation, it must keep the relation to the other methods:

msumZ Nil     === zero
msumZ (Zee _) === one
msumZ (pure a)  === msumZ (Cons a Nil) === a
