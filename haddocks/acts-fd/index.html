acts-fd

Type class for types equipped with semigroup action.

There are ambiguity commonly exists between actions. For example, a group acts on itself by multiplication or conjugation or some other way depending on its context.

Unlike already-published libraries for semigroup actions, this library aims to provide a way to disambiguate actions by newtype wrappers around the type acted on. It allows FunctionalDependencies from the type acted on to the type of semigroup, reducing gotchas of type inference and overlapping instances.
